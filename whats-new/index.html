Welcome and thanks for using Fluent UI

This is a workbench theme. That means that VS Code's UI is being heavily modified for aestethic purposes only. There's no intention to enhance or compete with the original look. Is merely an alternative.

Also, please bear in mind that this theme is considered an experiment, and therefore beta software, since there's no official support for this type of modification.

Features

  • Dynamic light/dark theme - Based on the syntax, the workbench will update the styles to match on the fly!
  • Compact mode - The new layout uses more white space (padding) than the vanilla, if real state is an issue, you can toggle between regular and compact modes via settings

Thank you.
